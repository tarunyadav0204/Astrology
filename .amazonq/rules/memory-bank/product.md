# Product Overview

## Purpose
AstroRoshni is a comprehensive Vedic astrology application that provides authentic astrological calculations, predictions, and personalized insights using traditional Indian astrology principles with Lahiri Ayanamsa system.

## Value Proposition
- Authentic Vedic astrology calculations using Swiss Ephemeris engine
- AI-powered personalized predictions and insights using Google Gemini
- Multi-platform accessibility (Web, iOS, Android)
- Traditional chart styles (North Indian, South Indian)
- Real-time transit tracking and analysis
- Comprehensive life domain analysis (career, health, wealth, marriage, education)

## Key Features

### Core Astrology Features
- Birth chart generation (Lagna/Rashi chart, Navamsa D9, all divisional charts D1-D60)
- Multiple Dasha systems (Vimshottari, Chara, Yogini, Kalachakra, Shoola)
- Real-time planetary transits with date picker
- Nakshatra analysis with pada calculations
- Yoga detection (Pancha Mahapurusha, Raja Yoga, Dhana Yoga, etc.)
- Planetary strength calculations (Shadbala, Ashtakavarga)
- House strength and signification analysis
- Planetary relationships and aspects (Vedic aspects, Jaimini aspects)

### Life Domain Analysis
- **Career Analysis**: Profession suitability, timing, leadership potential, work patterns
- **Health Analysis**: Disease susceptibility, body constitution, remedy suggestions
- **Wealth Analysis**: Financial prospects, wealth timing, trading luck
- **Marriage Analysis**: Compatibility (Guna Milan), timing, partner characteristics
- **Education Analysis**: Academic strengths, suitable fields, timing
- **Progeny Analysis**: Child prospects, timing, characteristics

### Advanced Features
- AI-powered conversational chat for personalized guidance
- Blank chart analysis (query-based predictions without birth details)
- Muhurat calculator for auspicious timing
- Panchang (daily almanac) with monthly view
- Festival calendar with regional variations
- Physical trait analysis and appearance predictions
- Nadi astrology integration
- KP (Krishnamurti Paddhati) system support
- Mundane astrology for world events
- Financial forecasting and market predictions

### User Experience Features
- Multi-language support
- Credit-based system for premium features
- User authentication and profile management
- Chat history and session management
- PDF report generation and sharing
- Responsive design for all devices
- Offline capability in mobile apps
- Admin dashboard for user management

## Target Users

### Primary Users
- Astrology enthusiasts seeking authentic Vedic predictions
- Individuals looking for life guidance and timing decisions
- People interested in self-discovery through astrology
- Users seeking marriage compatibility analysis
- Career seekers wanting professional guidance

### Secondary Users
- Professional astrologers using the platform for calculations
- Students learning Vedic astrology
- Researchers studying astrological patterns
- Spiritual seekers exploring Vedic sciences

## Use Cases

1. **Birth Chart Analysis**: Generate and analyze complete birth chart with all divisional charts
2. **Life Event Timing**: Determine auspicious timing for marriage, career changes, investments
3. **Compatibility Check**: Assess marriage compatibility using traditional Guna Milan
4. **Career Guidance**: Identify suitable professions and career timing
5. **Health Insights**: Understand health vulnerabilities and preventive measures
6. **Daily Guidance**: Check daily transits and panchang for decision-making
7. **Conversational Queries**: Ask specific questions and get AI-powered astrological insights
8. **Muhurat Selection**: Find auspicious times for important events
9. **Remedy Suggestions**: Get personalized remedies based on chart analysis
10. **Transit Tracking**: Monitor current planetary positions and their effects

## Business Model
- Freemium model with basic features available to all users
- Credit-based system for premium features (AI chat, detailed reports, advanced calculations)
- Promo codes and credit request system
- Admin-managed credit allocation
- Potential for subscription tiers (future enhancement)
